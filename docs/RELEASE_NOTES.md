# 多读书桌面版 v1.0.0 发布说明

**发布日期**: 2026年2月3日  
**版本号**: 1.0.0  
**状态**: 正式版 (GA)

---

## 📖 概述

多读书桌面版 v1.0.0 正式发布！

多读书是一款**本地优先（Local-First）且支持绿色便携（Portable）**的沉浸式英语学习工作站。旨在为英语学习者提供一个集阅读、查词、笔记、AI 助教于一体的高效学习环境。

无需复杂的安装配置，下载解压即可开始您的沉浸式英语学习之旅。

---

## ✨ 核心特性

### 🔥 全能阅读体验

#### 1. 三合一阅读器
- **PDF 阅读**: 基于 react-pdf，支持大文件秒开（数百 MB 轻松应对），流畅的翻页体验。
- **EPUB 电子书**: 基于 epub.js，完美呈现电子书排版，还原纸书般的阅读感受。
- **TXT 文本**: 独创的 column-based 分页渲染技术，优化中文与英文混合排版显示。

#### 2. 智能 AI 助教
- **模型自主掌控**: 支持接入任意兼容标准接口的 AI 模型，无论是云端最强模型还是本地私有模型，选择权完全在您手中。
- **意图识别**: 智能区分您的提问意图，自动在“语言学习助手”（翻译/语法解析）和“知识检索助手”之间切换。
- **上下文感知**: 深度集成的 FTS5 全文搜索，AI 回答问题时能引用书中原文，杜绝幻觉。
- **主动学习**: 根据当前阅读内容，智能生成启发式学习问题，帮助您检验理解程度。

#### 3. 极致词典系统
- **三级查询架构**: 缓存优先 → 本地词典 → AI 兜底，确保查词速度与准确性的完美平衡。
- **多词典聚合**: 支持导入 MDX 格式词典、集成 ECDICT 离线词典及在线词典源。
- **自定义顺序**: 支持拖拽调整不同词典的显示优先级，打造个人专属词典。
- **真人级发音**: 集成 Edge TTS，提供自然、清晰的单词与例句朗读。

#### 4. 闭环词汇管理
- **智能生词本**: 阅读时一键收藏生词，支持自定义标签分类管理。
- **语境例句**: 自动从阅读内容中捕获生词所在的句子作为例句，记忆更深刻。
- **科学复习**: 基于间隔重复记忆曲线（Spaced Repetition）的复习提醒，抗遗忘。

### 🚀 专为桌面打造

#### 1. 真正的便携版（Portable）
- **解压即用**: 下载 zip 包解压即运行，无需管理员权限，不写入注册表。
- **数据随身**: 所有用户数据（书籍、笔记、词典配置）均存储在程序目录下的 `data/` 文件夹。
- **U 盘模式**: 将整个文件夹复制到 U 盘，在任何 Windows 电脑上插上即用，学习进度无缝漫游。
- **零环境依赖**: 内置所有运行环境，无需安装 Python、Node.js 等运行时。

#### 2. 沉浸式交互
- **专注模式**: 阅读界面默认折叠无关边栏，提供纯净的阅读空间。
- **全局快捷键**: 支持系统级快捷键唤起查词或笔记，操作行云流水。
- **自动进度同步**: 实时保存阅读进度，下次打开自动跳转至上次阅读位置。
- **系统级集成**: 原生系统菜单支持，符合桌面应用操作习惯。

## 🛠️ 技术架构

多读书采用现代化的技术栈构建，兼顾开发效率与运行性能：

- **前端**: Next.js 16 (React 19) + Tailwind CSS 4，构建高性能、响应式的用户界面。
- **后端**: FastAPI + SQLAlchemy + SQLite，提供稳健的数据服务与本地存储。
- **桌面容器**: Electron 28，连接系统底层能力。
- **AI 引擎**: 灵活的 Supplier 架构，支持多种模型供应商。
- **语音**: Edge TTS，免费且高质量的语音合成服务。

## 📦 安装与使用

### 方式一：便携版（强烈推荐）⭐

最适合希望数据完全掌控、或需要在不同电脑间切换的用户。

1. 下载 `DuoDuShu-Desktop-Portable-v1.0.0.zip`。
2. 解压到任意目录（如 `D:\Tools\` 或 U 盘根目录）。
3. 双击 `多读书.exe` 即可运行。
4. **注意**: 所有数据将存储在解压目录下的 `data/` 文件夹中。

### 方式二：标准安装版

适合习惯传统安装方式的用户。

1. 下载 `DuoDuShu-Desktop-Setup-v1.0.0.exe`。
2. 双击运行，按照向导完成安装。
3. 从桌面快捷方式或开始菜单启动。

## ⚠️ 已知限制

作为 1.0.0 版本，我们在以下方面仍有优化空间：
- **复杂排版**: 极少数排版极其复杂的 PDF 或 EPUB 文件可能显示略有差异。
- **大词库搜索**: 当生词本积累超过数千词条时，本地全文检索速度可能略有下降。
- **平台支持**: 目前仅提供 Windows 版本 (x64)，其它版本有待开发。
