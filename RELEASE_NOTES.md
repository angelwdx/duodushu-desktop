# DuoDuShu Desktop v1.0.0 Release Notes

**发布日期**: 2026-01-31
**版本**: 1.0.0 (首个正式版本)

---

## 🎉 欢迎使用 DuoDuShu Desktop v1.0.0

多读书桌面客户端正式发布！这是一款**本地优先（Local-First）且支持绿色便携（Portable）**的沉浸式英语学习工作站。

---

## ✨ 核心功能

### 📖 阅读功能
- ✅ **高性能阅读** - 秒开数百 MB 的 PDF/EPUB 大文件
- ✅ **多格式支持** - 支持 PDF、EPUB、TXT 等常见电子书格式
- ✅ **词典集成** - 内置 ECDICT 英文词典，支持快速查词
- ✅ **自定义词典** - 支持导入和管理自定义词典

### 💾 数据管理
- ✅ **本地优先** - 所有数据存储在本地，无需上传云端
- ✅ **绿色便携** - 解压即用，数据随身携带
- ✅ **数据持久化** - 基于 SQLite 的本地数据库
- ✅ **便携模式** - 可整体复制到 U 盘或其他电脑使用

### 🔧 系统集成
- ✅ **跨平台支持** - Windows 版本首发（macOS/Linux 计划中）
- ✅ **离线支持** - 基础功能完全离线可用
- ✅ **现代化界面** - 基于 React 19 + Tailwind CSS 的响应式设计

---

## 📦 下载和安装

### 便携版（推荐）
- **文件**: `DuoDuShu-Desktop-Portable.exe` (142 MB)
- **优点**: 无需安装，解压即用，数据随身携带
- **使用方法**:
  1. 下载 `DuoDuShu-Desktop-Portable.exe`
  2. 双击运行
  3. 所有数据存储在 exe 同级的 `data/` 目录
  4. 可以整体复制到 U 盘或其他电脑使用

### 安装版
- **文件**: `Duodushu Setup 1.0.0.exe` (143 MB)
- **优点**: 标准 Windows 安装体验，支持卸载
- **使用方法**:
  1. 下载 `Duodushu Setup 1.0.0.exe`
  2. 双击运行安装程序
  3. 按照提示完成安装
  4. 数据存储在系统 AppData 目录

### 压缩包
- **文件**: `Duodushu-1.0.0-win.zip` (209 MB)
- **用途**: 备用下载方式

## 🖥️ 系统要求

- **操作系统**: Windows 7 或更高版本
- **处理器**: Intel/AMD 双核处理器或更高
- **内存**: 2 GB RAM 或更高
- **磁盘空间**: 300 MB 可用空间（便携模式）
- **网络**: 可选（基础功能完全离线）

## 🚀 快速开始

### 第一次使用
1. 下载并运行应用
2. 应用会自动创建数据目录
3. 导入你的电子书或使用内置词典
4. 开始学习！

### 导入电子书
1. 打开应用
2. 点击"导入"按钮
3. 选择 PDF、EPUB 或 TXT 文件
4. 文件会自动添加到你的库中

### 查询词汇
1. 在阅读时选中英文单词
2. 点击查询按钮或使用快捷键
3. 词典会显示单词的定义和例句

## 🐛 已知问题和限制

### 当前版本的已知限制
- 暂不支持 macOS 和 Linux（计划在 v1.1 中支持）
- 暂不支持文件关联（计划在 v1.1 中支持）
- 暂不支持自动更新（计划在 v1.1 中支持）

### 故障排查

**问题**: 应用无法启动
- **解决方案**: 检查 Windows 版本是否为 Windows 7 或更高，尝试重新下载应用

**问题**: 词典查询无法工作
- **解决方案**: 确保 `data/` 目录中存在 `ecdict.db` 文件

**问题**: 导入的电子书无法打开
- **解决方案**: 确保文件格式为 PDF、EPUB 或 TXT，文件大小不超过 1 GB

更多问题请查看 [故障排查文档](./docs/TROUBLESHOOTING.md)
